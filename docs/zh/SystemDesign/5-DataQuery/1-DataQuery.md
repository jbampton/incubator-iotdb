# 数据查询

数据查询有如下几种类型

* 原始数据查询
* 聚合查询
* 降采样查询
* 单点补空值查询
* 最新数据查询
* 按设备对齐查询
* 降采样补空值查询

为了实现以上几种查询，IoTDB 查询引擎中设计了针对单个时间序列的基础查询组件，在此基础上，实现了多种查询功能。
